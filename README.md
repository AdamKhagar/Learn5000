# Learn5000
Telegram bot to memorize words.
Bot name in the telegram @learn5000bot

# For whom this bot was designed?
This bot was created for Russian speakers to allow them to learn other languages. Maybe one day I will make support for multiple native languages.
# But 
but nothing prevents you from using the code of this project to adapt it to another language. To do this, you need to change message templates in the templates folder and add new dictionaries to the words folder.
